使用腾讯云对象存储搭建github pages图床

发表于 2018-09-04

Github Pages 是 Github 为开发者提供的静态页面网站服务，本站就是放在Github中的。此次教程给大家介绍下如何使用腾讯云为Pages图片加速。

点击 这里 先创建一个存储桶。存储桶是腾讯云对象存储的基本单位，一个存储桶有唯一的请求域名。访问权限建议选用公有读私有写，方便外网请求图片。

点击上传文件上传你想存储的图片。

腾讯云官方还提供了一个COS客户端，点击这里下载。

经测试，该客户端提供 拖拽上传、 在线复制文件链接等功能。
